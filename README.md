# v2tetotech
v2tetotech - универсальный комбайн для обхода локов по клиенту, User-Agent и HWID, а также анонимизации устройств и автопарсинге под клиенты по типу SFA/Mihomo. Сервер полностью берет работу на себя, благодаря чему работает обновление в клиентах, все одной ссылкой и особо не ощущается

## Зеркала
- [v2tetotech.onrender.com](https://v2tetotech.onrender.com) 🟢
## Подготовка
1. Установите Python3.9 и выше, на Windows обязательно Add to PATH
2. Склонируйте через `git clone` или скачайте репозиторий и перейдите в его папку 
3. Перейдите в папку репозитория

## Запуск сервера
1. Сначала, необходимо настроить `data/config.py` под ваши нужды, если вы запускаете сервер, выключайте debug, sensitive logging и стаьте flavor на release
2. Установите зависимости через `pip3 install -r requirements.txt`
3. Запустите сервер через `python3 main.py`

## Запуск бота
0. Убедитесь что сервер точно работает на localhost:8000 или прописаны рабочие сервера в bot/config.py!
1. Перейдите в папку bot/ и скопируйте .env.example под именем .env и заполните BOT_TOKEN
2. Установите зависимости через `pip3 install -r requirements.txt`
3. Запустите бота через `python3 bot.py`
